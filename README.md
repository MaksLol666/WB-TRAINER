# WB-TRAINER
Тренировки и тесты для пвз

## Настройка

1. Скопируйте пример переменных окружения:

   ```bash
   cp .env.example .env
   ```

2. Укажите токен Telegram-бота в `.env`:

   ```env
   BOT_TOKEN=ваш_токен_бота
   ```

3. Установите зависимости и запустите бота:

   ```bash
   pip install -r requirements.txt
   python main.py
   ```
